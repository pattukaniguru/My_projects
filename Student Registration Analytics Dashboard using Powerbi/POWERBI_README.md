# 📊 Student Registration Analytics Dashboard

A live Power BI dashboard developed to analyze and visualize student registration data collected through Google Sheets. The dashboard provides real-time insights, interactive filtering, and automated reporting for efficient data analysis.

---

## 📈 Project Overview

This project demonstrates how Power BI can be integrated with Google Sheets to create a dynamic analytics dashboard. The solution enables real-time monitoring of student registrations through interactive visualizations and Key Performance Indicator (KPI) metrics.

---

## 🚀 Features

- **Live Data Integration:** Seamless connection to Google Sheets for real-time updates.
- **Interactive Visualizations:** Clickable charts that cross-filter data dynamically.
- **Comprehensive Analysis:** Breakdown of registrations by course, department, and college.
- **Trend Monitoring:** Track registration volume over time to identify peak periods.
- **Dynamic Reporting:** Use slicers to filter data by specific criteria instantly.
- **Performance Metrics:** At-a-glance KPI cards summarizing key data points.
- **Detailed Records Table:** A complete view of individual student registration entries.

---
