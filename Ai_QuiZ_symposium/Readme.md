🧠 AI Quiz Web Application



This project is a web-based quiz application designed to conduct AI/Technology-related quizzes with image-based questions, automatic evaluation, and dashboard support. It uses HTML, JavaScript, and Google Apps Script for quiz handling and response storage.



📌 Project Overview



The AI Quiz Application allows users to:



Attempt multiple-choice quiz questions



View questions with images and visual options



Submit responses through a web interface



Store and process quiz data using Google Apps Script



Manage quiz content through editable JavaScript files



View quiz results via a dashboard page

🚀 Features Implemented

✅ Quiz Interface



User-friendly quiz UI using HTML



Multiple-choice questions



Supports image-based questions and options



✅ Quiz Data Handling



All questions and answers are stored in quiz\_data.js



Easy to modify or extend quiz questions



✅ Google Apps Script Integration



google\_script\_code.js handles:



Form submission



Sending responses to Google Sheets



Backend processing without a server



✅ Dashboard Module



Separate dashboard page



Used for monitoring quiz data or results



Uses its own quiz\_data.js and Apps Script logic



✅ Media Support



Image assets stored in src/



Used for:



Visual questions



Option images



Logo-based and identification questions
